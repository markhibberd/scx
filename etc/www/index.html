scx

Version 0.1

scx is a scala compilation utility with support for cross compilation.

quick start

Here are the basic operation you can perform with scx.

Compile scala only: 

    scx -cp lib src classes

Compile scala and java: 

    scx -j -cp lib src classes

installation

scx is a single shell script. Just put it on your path and you are done.
